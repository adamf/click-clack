SPI bus
-------
   ESP32-S3        OLED #1 (Player 1)         OLED #2 (Player 2)
   GPIO12 SCK ───┬── SCK                  ┬── SCK
   GPIO11 MOSI ──┤── MOSI (D1/SDA)        ┤── MOSI (D1/SDA)
                 │                        │
   GPIO10 CS  ───┘── CS                   │
   GPIO13 CS  ──────────────────────────  ┘── CS
   GPIO9  DC  ───┬── DC                   ┬── DC
   GPIO8  RST ───┴── RST                  ┴── RST


Buttons (all to GND, MCU uses internal pull-ups)
------------------------------------------------
   GPIO4  ──[ MX P1 ]── GND      (left player paddle)
   GPIO5  ──[ MX P2 ]── GND      (right player paddle)
   GPIO6  ──[ MX MODE ]── GND
   GPIO7  ──[ MX SET  ]── GND
   GPIO15 ──[ MX UP   ]── GND
   GPIO16 ──[ MX DOWN ]── GND
   GPIO17 ──[ MX CENTER ]── GND  (small reset/pause)


Buzzer
------
   GPIO18 ──[ 220Ω ]──┬── piezo + ── GND
                      │
                   (or gate of 2N7000 with piezo to 5V for louder)


Battery sense
-------------
   BAT+ ──[ 100kΩ ]──┬──[ 100kΩ ]── GND
                     │
                     └── GPIO1 (ADC1_CH0)
   max input ≈ 2.1V  (4.2V / 2)
